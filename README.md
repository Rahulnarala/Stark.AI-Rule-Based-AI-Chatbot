# Stark.AI

Stark.AI is a rule-based AI chatbot developed to demonstrate the fundamentals of Artificial Intelligence using predefined rules and logical decision-making. Unlike machine learning chatbots, Stark.AI responds to user queries by analyzing keywords and matching them with predefined conditions using `if-else` logic. The chatbot provides fast, consistent, and reliable responses for supported queries while maintaining a continuous conversational experience through an interactive web interface.

## Key Features

* Rule-based chatbot built using conditional (`if-else`) logic
* Interactive and user-friendly web interface
* Handles greetings, general queries, and exit commands
* Continuous conversation loop for seamless interaction
* Keyword-based intent recognition and response generation
* Fast and lightweight with no external AI models required
* Easily expandable by adding new rules and responses
* Responsive design compatible with desktop and mobile devices

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## How It Works

1. The user enters a message.
2. Stark.AI processes the input by identifying keywords.
3. The chatbot compares the input against predefined rules.
4. If a matching rule is found, the corresponding response is displayed.
5. If no matching rule exists, a default response is returned.

## Future Enhancements
